El usuario debe ver primero:

Títulos.
Descripción.
Botones.
Servicios.
Llamadas a acción.

El código solamente debe aportar ambiente tecnológico.

Fondo de código estilo ambiente tecnológico

Crear un fondo inspirado en un editor de código, pero extremadamente ligero.

Debe parecer que detrás de la página existe un entorno de desarrollo funcionando.

Características:

Código desenfocado.
Movimiento lento.
Escritura automática ocasional.
Cursor sutil.
Líneas de código flotando.
Partículas pequeñas.
Glow tecnológico.

NO debe parecer un bloque de texto leyendo código.

Opacidad y colores EXACTOS

Fondo principal:

#0A2540

Texto principal:

#F5F5F7

Texto secundario:

#A8B5D1

El código del fondo debe tener:

Código azul/cyan:

Código violeta:

Texto secundario:

rgba(245,245,247,0.08)

Posición del fondo

Debe estar:

Absolutamente detrás del contenido.
position:absolute.
z-index menor que el contenido.
No ocupar espacio del layout.
No modificar tamaños de secciones.

Ejemplo:

.hero-background {
 position:absolute;
 inset:0;
 z-index:0;
 pointer-events:none;
 opacity:0.12;
}

.hero-content {
 position:relative;
 z-index:2;
}
Blur obligatorio

Aplicar profundidad:

filter: blur(3px);

En algunas zonas:

filter: blur(8px);

El usuario debe percibir:

"hay código detrás"

NO:

"estoy leyendo código".
